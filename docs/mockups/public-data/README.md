# Viešų medžioklės duomenų pateikimo prototipas (issue #218)

Savarankiškas HTML pavyzdys, kaip galėtų atrodyti perdarytas viešas
`medziokle.biip.lt/zemelapis` (lentelė + žemėlapis).

- Lentelės skaičiai — **realūs** (prod, 2025/2026 sezonas).
- Žemėlapio pasiskirstymai tarp savivaldybių — **iliustraciniai** (nacionalinės sumos realios),
  nes savivaldybės id→pavadinimas mapping'o nėra medžioklės DB.

Atidaryti: atsidaryti `medziokle-viesi-duomenys-prototipas.html` naršyklėje.
